## 프로젝트 목표
- 정보 접근 효율화: 공지사항 및 정보를 쉽게 검색하고 빠르게 찾을 수 있는 시스템 구축한다.
- 자동 응답 시스템 개발: RAG와 LLM을 활용해 질문에 자동으로 정확한 답변 제공한다.
- 정보 불균형 해소: 신입생 등 정보 접근이 어려운 학생들에게 필요한 정보 제공한다.
